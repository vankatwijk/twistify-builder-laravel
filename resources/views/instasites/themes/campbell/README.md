# Campbell Theme

The Campbell theme is designed to provide a clean and modern look inspired by the Craig Campbell SEO website. This theme is built for Laravel and follows best practices for structure and organization.

## Installation

1. **Clone the repository** or download the theme files.
2. **Place the theme** in the appropriate directory within your Laravel application, typically under `resources/views/instasites/themes/`.
3. **Run the necessary migrations** and seeders if applicable.

## Features

- Responsive design that adapts to various screen sizes.
- Custom styles and scripts tailored for the Campbell theme.
- Modular structure with reusable components and partials for easy maintenance.
- SEO-friendly layout and design.

## File Structure

- **page.blade.php**: Main template for displaying pages.
- **post.blade.php**: Template for individual blog posts.
- **assets/**: Contains CSS, JS, and fonts used in the theme.
  - **css/**: Main styles and additional styles.
  - **js/**: Custom scripts for interactivity.
  - **fonts/**: Font files and usage instructions.
- **layouts/**: Base layout structure for the theme.
- **partials/**: Various sections of the theme, including header, footer, and content sections.
- **components/**: Reusable components like cards and buttons.
- **theme.json**: Configuration settings for the theme.
- **README.md**: Documentation for the theme.

## Usage

To use the Campbell theme, ensure that your Laravel application is set up correctly and that the theme files are in the correct location. You can then reference the theme in your views and customize it as needed.
